MERN Job Portal App
A full-stack Job Portal application built with MongoDB, Express.js, React.js, and Node.js.
Users can post jobs, apply for jobs, and manage applications.

🚀 Features
User authentication (Signup/Login)
Post and manage job listings
Apply for jobs
Admin & user dashboards
Responsive UI

📂 Project Structure
root/
 ├── backend/   # Node.js + Express + MongoDB API
 └── frontend/  # React.js client

2️⃣ Backend Setup

cd backend
npm install
npm run dev 


3️⃣ Frontend Setup
Open a new terminal:

cd frontend
npm install
npm start

⚙️ Environment Variables
Create a .env file in backend/ and add:
PORT=
MONGO_URI=

# cloudinary
CLOUD_NAME=
API_KEY=
API_SECRET=

# jsonwebtoken
SECRET_KEY=
